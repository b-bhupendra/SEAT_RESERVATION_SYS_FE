## Key Features
- **Smart Seating**: Intelligent assignments for monthly or flexible durations.
- **Fluid Payments**: Seamless integrated tracking for all your transactions.
- **Premium Design**: Modern fluid UI with Framer Motion and glassmorphism.

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```sh
   npm install
   ```
2. Create your environment variables file by copying the example:
   ```sh
   cp .env.example .env
   ```
   *(Provide `GEMINI_API_KEY` and `APP_URL` if needed)*
3. Run the development server:
   ```sh
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
